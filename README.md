# 📊 AI Job Market Analysis Dashboard (Power BI)

## 🚀 Overview
This project presents an interactive AI Job Market Dashboard built using Power BI.  
It analyzes global job trends, salary insights, company growth, and geographic distribution.

---
## 🎯 Problem Statement
The AI job market is growing rapidly, but insights are scattered.  
This dashboard helps in:
- Understanding job trends  
- Analyzing salary distribution  
- Tracking company growth  
- Exploring global opportunities  

---
## 🛠️ Tools Used
- Power BI  
- DAX
